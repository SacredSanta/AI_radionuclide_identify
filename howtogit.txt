중간에 commit이나,
웹상으로 따로 수정해서 꼬이는 오류가 생겼다면..
(1) git pull origin main    : 다시 땡겨줘야됨
(2) 만약 땡길라하는데 rebase 관련 문제가 발생시
    git config pull.rebase true
    git rebase --continue
(3) 다시 git pull origin main
(4) 그 후 commit 진행
